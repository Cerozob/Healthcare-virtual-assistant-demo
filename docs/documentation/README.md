# Documentation

Core project documentation for the AWSomeBuilder healthcare AI system for AnyCompany Healthcare.

## Files

### Project Specifications

- **`notes_spec.md`** - Main project specification document (in Spanish)
  - Problem statement and solution overview
  - Use case description for AnyCompany healthcare
  - Project components and requirements

### Infrastructure Documentation

- **`infrastructure-setup.md`** - Complete infrastructure setup guide
  - CDK stack documentation
  - AWS service configurations
  - Deployment instructions
  - Security and compliance information

### Compliance Documentation

- **`cdk-nag-suppression.md`** - CDK Nag security suppression documentation
  - Development phase security approach
  - Suppressed rules and rationale
  - Compliance transition strategy

## Language Note

The main specification (`notes_spec.md`) is written in Spanish as it was created for a Spanish-speaking healthcare company (AnyCompany).
