# Architecture Documentation

System architecture diagrams and design documents for the AWSomeBuilder 2 healthcare AI system for AnyCompany Healthcare.

## Files

### Architecture Diagrams

- **`AWSomeBuilder2-Planned.png`** - Current planned architecture diagram (PNG format)
- **`AWSomeBuilder2-Planned.pdf`** - Current planned architecture diagram (PDF format)
- **`AWSomeBuilder2_draft.pdf`** - Draft version of the architecture diagram

### Editable Diagrams

- **`AWSomeBuilder2.drawio`** - Draw.io source file for architecture diagrams
- **`.$AWSomeBuilder2.drawio.bkp`** - Backup of the Draw.io source file

## Usage

The Draw.io file can be opened with:

- [Draw.io web application](https://app.diagrams.net/)
- Draw.io desktop application
- VS Code with Draw.io extension

## Versions

- **Draft**: Initial architecture concepts and ideas
- **Planned**: Current target architecture for implementation

Both versions are maintained to track the evolution of the system design.
