# Test Files

Test scripts, audio files, and other testing resources used during development of the AWSomeBuilder 2 healthcare AI system for AnyCompany Healthcare.

## Files

### Audio Test Files

- **`test_speech_esmx.mp3`** - Spanish AnyCompany Healthcare consultation audio sample
  - Used for testing Amazon HealthScribe speech-to-text functionality
  - Contains AnyCompany Healthcare conversation in Spanish
  - Serves as input for AnyCompany Healthcare transcription and summarization testing

### Test Scripts

- **`text_speech_script.md`** - Spanish AnyCompany Healthcare consultation script
  - Text version of the AnyCompany Healthcare consultation dialogue
  - Used for testing and validation of transcription accuracy
  - Contains typical AnyCompany Healthcare consultation phrases and terminology in Spanish

## Usage

### Audio Testing

The MP3 file can be used to test:

- Amazon HealthScribe AnyCompany Healthcare transcription
- Amazon Transcribe Medical speech-to-text
- Audio processing pipelines
- AnyCompany Healthcare terminology recognition

### Script Validation

The text script serves as:

- Ground truth for transcription accuracy testing
- Reference for AnyCompany Healthcare terminology validation
- Template for creating additional test scenarios

## Language

All test files are in Spanish to match the target healthcare environment for AnyCompany Healthcare.
