# Sample Data

Healthcare sample data for development and testing.

## Medical Records (`/medical-records/`)

- `historiamedica.pdf` - Standard medical history document
- `CopiaHistoriaClinica.pdf` - Password-protected medical record copy

## Spanish Templates

- `protocolos-medicos-espanol.json` - Medical protocols with specialties, symptoms, exams
- `plantillas-laboratorio-espanol.json` - Lab result templates with Spanish terminology
- `proveedores-departamentos-espanol.json` - Medical providers and hospital departments

## AWS AI Service Samples (`/aws-services/`)

Sample outputs from AWS AI services:
- **BDA**: Bedrock Data Automation document processing results
- **HealthScribe**: Clinical conversation summaries and transcripts
- **Transcribe Medical**: Speech-to-text processing samples
- **Textract**: Document text extraction results

All medical records use anonymized or fictional patient data for privacy compliance.
