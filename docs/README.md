The sample data includes examples from:

- **Bedrock Data Automation (BDA)**: Document processing results
- **HealthScribe**: AnyCompany Healthcare conversation transcription and summarization
- **Transcribe Medical**: AnyCompany Healthcare speech-to-text processing
- **Textract**: AnyCompany Healthcare document text extraction
- **Healthcare Records**: Sample AnyCompany Healthcare patient records in Spanish

These samples are used for development, testing, and demonstration of the AI-powered healthcare solution for AnyCompany Healthcare.
